# Roadside Assistance Web

A business website for a roadside assistance company, built with React, TypeScript, and Vite, and deployed via AWS Amplify.

The site gives customers quick access to contact information, service rates, and a description of the business — designed to be fast, mobile-friendly, and easy to maintain.

## Live Site
[View the live site](https://main.d1x2q8nzsp978o.amplifyapp.com/)

## Features
- Contact information and call-to-action for customers needing immediate help
- Service rates and offerings
- Business description and overview
- Responsive design for mobile and desktop

## Tech Stack
- React 18
- TypeScript
- Vite
- CSS
- AWS Amplify (hosting)

## Getting Started
```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build
```
